# **DOTNET**



##### **Aula 02** - Lógica do C# 





* ***C#*** semelhante ao Java, é uma linguagem de programação tipada sendo necessária colocar o tipo de variável (string, int, float etc) e sempre fechar com ";"
* Sua estrutura de controle de condicional e repetição também segue a mesma lógica do Java;
* Console.WriteLine() - Saída no console;
* Consolte.ReadLine() - Entrada no console;



**OBS.: Console.WriteLine() só retorna como string, caso necessite que seja um int ou double precisa usar a função Convert.To**
